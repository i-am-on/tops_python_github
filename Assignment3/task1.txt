Task1. What is File function in python? What is keywords to create and write file.

Ans. The key function for working with files in Python is the open() function. 
The open() function takes two parameters; filename, and mode.

-To write to an file, you must add a parameter to the open() function:

"w" - Write - will overwrite any existing content

"a" - Append - will append to the end of the file
