Que.How will you remove last object from a list? 
Suppose list1 is [2, 33, 222, 14, and 25], what is list1 [-1]?


*To remove the last object from the list user can uses 
  pop function without giving any specific index.

list1 = [2,33,222,14,25]
list1.pop()
print(list1)

-In above list -1 represents negetive index so the answer will be 25.
 because, value on -1 index is 25.
